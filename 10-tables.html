Contextual Classes

Contextual classes can be used to color the whole table (<table>), the table rows(<tr≶) or table cells (<td≶). The classes that can be used are: .table-primary, .table-success, .table-info, .table-warning, .table-danger, .table-active, .table-secondary, .table-light and .table-dark:

Default Defaultson john@example.com
Primary Moe mary@example.com
Success Dooley july@example.com
Danger Dooley july@example.com
Info Dooley july@example.com

Active Dooley july@example.com
